# {{PROJECT_NAME}}

A PC project created with [create-shy]({{TEMPLATE_NAME}}) template.

## Features

- Vue 3 + TypeScript + Vite
- Element Plus UI Framework
- Pinia State Management
- Vue Router
- Axios HTTP Client
- ESLint + Prettier

## Getting Started

\`\`\`bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build

# Preview production build
pnpm preview
\`\`\`

## Project Structure

\`\`\`
src/

├── stores/      # Pinia stores
├── views/       # Page views
├── App.vue      # Root component
└── main.ts      # Entry point
